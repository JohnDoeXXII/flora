Godot
-------

&nbsp;  When writing GDScript, always use static typing — annotate all variables (var x: int), function parameters, and

&nbsp;  return types. Prefer class\_name on any script that defines enums or constants intended for use in other scripts.

&nbsp;  Never leave type annotations off to work around a tooling issue; fix the root cause instead.
